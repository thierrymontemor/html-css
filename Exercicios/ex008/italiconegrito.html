Formatações

Negrito / Destaque

Para o uso correto do negrito, adiciona-se a tag 'strong' nos termos desejados.

Itálico / Ênfase

Ja para o uso do ítalico, usa-se a tag 'em' nos termos desejados.

Texto marcado

Para marcar um texto é só inserir a tag 'mark'. Se quiser inserir outra cor, usa-se CSS e deve-se inserir o parâmetro'style="background-color: [cor];" ' dentro da tag 'mark' e selecionar a cor desejada.
Detalhe: para mudar a cor de todas as marcações de uma página, deve-se usar a tag com CSS na parte do 'head'.
